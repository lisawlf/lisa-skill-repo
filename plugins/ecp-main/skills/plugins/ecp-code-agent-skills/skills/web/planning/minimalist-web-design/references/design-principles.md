# Reference: Minimalist Design Principles (2026 Standard)

Domain knowledge for the design and audit workflows. Covers core philosophy,
composition, typography, color, and assets.

## 1. Core Philosophy & Conception

### Avoid "Visual Recycling"
Do not use classic SaaS/B2B templates — such as a large hero image beside a vague
headline and three feature cards. The goal is unmistakable individuality and a
handmade, editorial feel. The fastest tell of a recycled design is that it could
belong to any company in any industry.

### Dual Moodboarding
Separate research strictly into two boards:
- **"Feel" moodboard** — aesthetics, typography, inspiration.
  Sources: **Awwwards**, **Site Inspire**.
- **"Performance" moodboard** — conversion, UI patterns.
  Sources: **Mobbin**, **Land-book**.

Keeping them separate prevents pretty-but-ineffective work and effective-but-generic work from contaminating each other.

### Layered Borrowing
Never copy whole websites. Isolate the layers **layout, typography, color, and
motion**, and borrow each deliberately and function-specifically. Originality
comes from recombining well-chosen layers, not from cloning a single source.

## 2. Visual Composition & Layout

### Whitespace as the Foundation
Use extremely generous and consistent spacing. Whitespace is not empty space — it
is an active design element that directs attention and reduces cognitive load.

### One Concept per Screen
Reduce each visible area (viewport) to a single, crystal-clear message. Do not
overload the user with competing elements within one screen.

### Asymmetry & Editorial Design
Bring visual tension into the layout with **asymmetric grids, F-patterns,
Z-patterns, or triangular compositions**. Predictable, perfectly symmetric grids
read as generic.

### Brutalist Minimalism
Dare to use raw, heavily grid-based designs with visible edges that foreground the
content without excessive decoration.

## 3. Typography & Color

### Extreme Hierarchy
Rely on drastic scale jumps. Use enormous, poster-like ("oversized") headlines in
direct contrast to small, perfectly legible body text. The contrast itself carries
the design.

### Font Limitation
Limit to a maximum of **2–3 typeface families**, favoring modern, very clean
sans-serif fonts.

### Color Strategy
Limit to **2–3 primary colors**. To break up the reduced aesthetic, use
**"maximalist color systems" (dopamine hues)** — highly saturated, almost garish
colors for very specific accent surfaces, rather than classic, soft gradients.

## 4. Assets & Illustrations

### No Stock Photography
Use authentic, professional images or purpose-built custom illustrations that
strengthen brand character.

### Video Restrictions
Avoid compute-heavy background videos (`.mp4`), which ruin performance. Use
lightweight `.webm` formats instead, or generate backgrounds programmatically via
WebGL (e.g. animated Bayer dithering).

### Asset Optimization
Optimize every image losslessly into modern formats like **WebP** or **AVIF**, and
deliver them at their exact native display dimensions.
