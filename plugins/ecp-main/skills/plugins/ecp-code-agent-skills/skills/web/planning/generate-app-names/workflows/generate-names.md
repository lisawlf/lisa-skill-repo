# Workflow: Generate Names

<required_reading>
**Read these reference files NOW:**
1. references/naming-strategies.md
2. references/linguistic-principles.md
3. references/ideation-methods.md
</required_reading>

<process>

## Step 1: Requirements Gathering

Gather essential information about the product. If not provided, ask:

**Core questions:**
- What does this product/app do? (1-2 sentence description)
- Who is the target user? (demographics, psychographics)
- What is the core benefit or value proposition?
- What tone/personality should the name convey? (professional, playful, premium, approachable, innovative, trustworthy)

**Optional but valuable:**
- What are 3-5 competitor names in this space?
- Are there any words or sounds that MUST be avoided?
- Is this for a global audience or specific region?
- Any existing brand elements the name must work with?

## Step 2: Strategic Analysis

Based on gathered information, determine:

**Naming Strategy Selection:**
Use the strategy taxonomy from `naming-strategies.md` to recommend 1-2 approaches:
- B2B/functional → Consider descriptive or suggestive
- Consumer/lifestyle → Consider suggestive or evocative
- Premium/luxury → Consider evocative
- Global/innovative → Consider abstract or hybrid
- Tech/digital → Consider hybrid/compound or suggestive

**Sound Profile Selection:**
Use `linguistic-principles.md` to identify target phonetic qualities:
- Strength/energy needed → Plosives (k, p, t, b, g)
- Warmth/comfort needed → Fricatives/nasals (m, n, l, s)
- Sophistication needed → Long vowels (/aɪ/, /eɪ/)
- Playfulness needed → Repetitive vowels (/uː/)

Document the strategic direction before ideation.

## Step 3: Divergent Ideation

Generate 30-50+ raw name candidates using multiple techniques:

**Technique rotation (spend 2-3 minutes each):**

1. **Direct association:** List 10 words related to the product's function, benefit, emotion
2. **Mind mapping:** Branch from core concept to find unexpected connections
3. **SCAMPER:** Apply each lens to top 5 seed words
4. **Portmanteau creation:** Combine relevant concept pairs
5. **Sound-first generation:** Create words that SOUND right for the brand profile
6. **Foreign language mining:** Check translations of key concepts in Latin, Greek, Japanese
7. **Prefix/suffix exploration:** Add tech suffixes (-ly, -ify, -io, -hub) to core words

**Quantity over quality at this stage.** Do not self-censor.

## Step 4: First Filter

Reduce candidates to 15-20 using basic criteria:

- [ ] Can you pronounce it on first try?
- [ ] Can you spell it after hearing it once?
- [ ] Is it 2-3 syllables? (prefer)
- [ ] Does it avoid obvious negative associations?
- [ ] Is it distinct from major competitors?

Remove obvious failures but keep anything promising.

## Step 5: SMILE/SCRATCH Evaluation

Score remaining candidates against the framework:

**SMILE (1-5 points each, higher is better):**
| Criterion | Score Guide |
|-----------|-------------|
| **Simple** | 5 = immediately clear, 1 = confusing spelling/pronunciation |
| **Meaningful** | 5 = instant brand connection, 1 = arbitrary |
| **Imaginative** | 5 = unique and fresh, 1 = generic or overused |
| **Lovable** | 5 = creates delight, 1 = no emotional response |
| **Elegant** | 5 = refined and appropriate, 1 = clunky or crude |

**SCRATCH (deduct 1-3 points each if applicable):**
- Spelling challenged: -3 if often misspelled
- Copycat: -3 if too similar to competitor
- Restrictive: -2 if limits future growth
- Annoying: -2 if grating pronunciation or meaning
- Tame: -1 if forgettable
- Curse of knowledge: -2 if only insiders understand
- Hard to pronounce: -3 if stumbles in conversation

**Calculate net scores.** Top 5-8 candidates advance.

## Step 6: Strategic Fit Assessment

For top candidates, evaluate strategic alignment:

- Does it support the intended brand personality?
- Does it work with potential taglines?
- Can you envision it in a logo?
- How does it sound in a sentence? ("Have you tried [name]?")
- Does it scale for potential product extensions?

## Step 7: Preliminary Availability Check

For top 3-5 candidates, note:

**Domain considerations:**
- .com availability (most valuable)
- Alternative TLDs (.io, .co, .app, .ai)
- Potential to acquire if taken

**Social handle availability:**
- Check if @name or @getname or @tryname are available on major platforms

**Trademark considerations:**
- Quick search for exact matches in the product category
- Note any potential conflicts for legal follow-up

## Step 8: Final Presentation

Present top 3-5 names with:

1. **The name**
2. **Strategy type** (suggestive, evocative, hybrid, etc.)
3. **Meaning/rationale** (why it works for this product)
4. **Phonetic analysis** (what sounds communicate)
5. **SMILE score** (total out of 25)
6. **Potential concerns** (any SCRATCH issues or availability notes)
7. **Recommended tagline pairing** (optional)

Rank by overall strategic fit, not just SMILE score.

</process>

- [ ] 30+ raw candidates generated

- [ ] Top 3-5 presented with full strategic analysis
- [ ] Availability considerations noted
</success_criteria>
