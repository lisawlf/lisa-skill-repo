<overview>
Creative name generation requires moving beyond standard vocabulary into new linguistic territories. Structured brainstorming methods help generate innovative options while avoiding common traps of conventional thinking.
</overview>

<core_principle>
In the ideation phase, **quantity precedes quality**. Generate a high volume of ideas to bypass self-censorship. Evaluation comes later. The goal is 50-100+ raw candidates before any filtering.
</core_principle>

<analytical_techniques>

<technique name="mind_mapping">
<description>
A visual method starting with a central concept and branching into related thoughts, revealing unexpected patterns and connections.
</description>

<process>
1. Place the core product concept in the center
2. Branch out with primary associations (features, benefits, emotions, users)
3. Extend each branch with secondary associations
4. Look for unexpected connections between distant branches
5. Extract naming candidates from compelling intersections
</process>

<example>
**Central concept:** Project management app

**Branches:**
- Organization → folders, filing, order, clarity, structure
- Collaboration → together, team, sync, harmony, orchestra
- Progress → forward, momentum, velocity, milestone, journey
- Control → command, helm, captain, pilot, navigator

**Intersection insight:** "Navigator" + "team harmony" → names like **Compass**, **Waypoint**, **Helmteam**
</example>
</technique>

<technique name="brainwriting">
<description>
A silent, anonymous process where participants write ideas and pass them for expansion. Prevents dominant voices from steering the group.
</description>

<process>
1. Each participant writes 3 name ideas on paper (2 minutes)
2. Pass papers to the next person
3. Read existing ideas and add 3 more inspired by them (2 minutes)
4. Repeat 4-6 rounds
5. Collect all papers and compile candidates
</process>

<solo_adaptation>
For individual brainstorming:
1. Write 5 names quickly
2. Set a timer for 5 minutes and do something else
3. Return and write 5 more inspired by (but not identical to) the first batch
4. Repeat 3-4 times
</solo_adaptation>
</technique>

<technique name="reverse_brainstorming">
<description>
Instead of solving the naming challenge, identify ways to make the name worse, then reverse those insights.
</description>

<process>
1. Ask: "What would make the worst possible name for this product?"
2. List terrible name characteristics (hard to spell, offensive meaning, similar to competitor, etc.)
3. Reverse each bad characteristic into a positive requirement
4. Generate names that fulfill the positive requirements
</process>

<example>
**Bad characteristics:**
- Impossible to pronounce → Requirement: 2 syllables, common sounds
- Sounds like a medical condition → Requirement: Positive, energetic associations
- Already used by 50 companies → Requirement: Unique, invented element

**Reversed insight leads to:** Short, positive, coined names
</example>
</technique>

<technique name="starbursting">
<description>
Question-driven exploration using the six-point star: Who, What, Where, When, Why, How.
</description>

<questions>
- **WHO** uses this product? What are they called? What do they call themselves?
- **WHAT** does it do? What problem does it solve? What feeling does it create?
- **WHERE** is it used? What environments? What physical or digital spaces?
- **WHEN** is it used? What moments? What triggers usage?
- **WHY** do people choose it? What motivates them? What do they aspire to?
- **HOW** does it work? What's the mechanism? What's the transformation?
</questions>

<name_extraction>
Each answer contains potential naming directions:
- WHO answers → user identity names (Creator, Pioneer, Maker)
- WHAT answers → benefit names (Clarity, Momentum, Shield)
- WHERE answers → environment names (Studio, Hub, Forge)
- WHY answers → motivation names (Ambition, Quest, Purpose)
</name_extraction>
</technique>

</analytical_techniques>

<scamper_framework>

<overview>
SCAMPER modifies existing ideas through seven specific lenses. Apply each lens to seed words related to your product.
</overview>

<lens name="substitute">
Replace a word, syllable, or letter with another.

**Examples:**
- "Quick" → "Quik" (spelling substitution)
- "Light" → "Lite" (phonetic substitution)
- "Transfer" → "Transfyr" (vowel substitution)
</lens>

<lens name="combine">
Merge two distinct concepts (portmanteaus and compounds).

**Examples:**
- Net + Flicks → Netflix
- Pin + Interest → Pinterest
- Micro + Soft → Microsoft
</lens>

<lens name="adapt">
Adjust a word for a different context or meaning.

**Examples:**
- "Safari" (African trip → browser exploration)
- "Amazon" (river → endless commerce)
- "Apple" (fruit → technology)
</lens>

<lens name="modify_magnify">
Change spelling, sounds, or emphasis for impact.

**Examples:**
- "Light" → "LYFT" (capitalization + spelling)
- "Fone" instead of "Phone"
- "Krispy" instead of "Crispy"
</lens>

<lens name="put_to_other_use">
Take a word from an unrelated industry or domain.

**Examples:**
- "Slack" (engineering term → workplace communication)
- "Buffer" (computing term → social media tool)
- "Pipeline" (oil industry → sales process)
</lens>

<lens name="eliminate">
Shorten by removing letters, syllables, or sounds.

**Examples:**
- "Application" → "App"
- "Photograph" → "Photo" → "Pho"
- "Refrigerator" → "Fridge"
</lens>

<lens name="reverse_rearrange">
Flip structure, order, or spelling.

**Examples:**
- Reversing word order: "BookFace" → "Facebook"
- Anagram exploration: "Listen" contains "Silent"
- Suffix → Prefix: "-ify" → "Fy-" (Spotify → Fyspot)
</lens>

</scamper_framework>

<portmanteau_techniques>

<overview>
Portmanteaus blend sounds and meanings of two or more words to create proprietary terms. The audience engages as they "decode" the blend.
</overview>

<blend_types>

<type name="start_plus_end">
Initial segment of word A + final segment of word B.

**Mechanics:** Take the beginning of the first word and the ending of the second.

**Examples:**
- Br(eakfast) + (l)unch = **Brunch**
- Net(work) + (fl)icks = **Netflix**
- Sm(oke) + (f)og = **Smog**
- Pin(ning) + (int)erest = **Pinterest**
</type>

<type name="start_plus_start">
Initial segments of both words combined.

**Examples:**
- Micro + Software = **Microsoft**
- Romantic + Comedy = **Romcom**
- Situation + Comedy = **Sitcom**
- Binary + Digit = **Bit**
</type>

<type name="overlapping">
Common sounds or letters shared by both words fuse together.

**Examples:**
- Pin + Interest (shared "in") = **Pinterest**
- Picture + Dictionary (shared "ict") = **Pictionary**
- Guess + Estimate (shared "es") = **Guesstimate**
- Motor + Hotel (shared "ot") = **Motel**
</type>

<type name="partial_splinter">
One full word + a fragment (splinter) from another.

**Examples:**
- Fan + Magazine = **Fanzine**
- Work + Alcoholic = **Workaholic**
- Choco + Alcoholic = **Chocoholic**
- Web + Seminar = **Webinar**
</type>

</blend_types>

<creation_process>
1. List two key concepts/words for your product
2. Break each into syllables and phonemes
3. Identify potential fusion points (overlapping sounds, complementary endings)
4. Generate 10+ blend variations
5. Test for pronounceability and meaning clarity
6. Verify the blend doesn't have unintended meanings
</creation_process>

</portmanteau_techniques>

<word_manipulation_tools>

<category name="prefix_suffix_exploration">
Add common prefixes and suffixes to core concept words:

**Prefixes:** un-, re-, pre-, post-, pro-, anti-, neo-, meta-, hyper-, ultra-, super-, multi-, omni-

**Suffixes:** -ly, -ify, -ize, -able, -ful, -less, -ness, -ment, -tion, -ity, -io, -ia, -ify, -hub, -ly, -box, -base

**Tech-specific:** -ly, -ify, -io, -app, -hub, -stack, -kit, -labs, -works
</category>

<category name="letter_manipulation">
- Remove vowels: Flickr, Tumblr, Scribd
- Double letters: Wii, Digg, Fiverr
- Replace letters: Lyft (Lift), Tru (True)
- Add letters: Shopify (Shop + ify)
</category>

<category name="foreign_language_mining">
Explore translations of key concepts:
- Latin roots for gravitas
- Greek for technical/scientific feel
- Japanese for minimalist/design aesthetic
- Nordic languages for strength/nature
</category>

</word_manipulation_tools>

<ideation_session_structure>

<phase name="warm_up" duration="5 minutes">
Free association with no judgment. Write every word that comes to mind related to the product.
</phase>

<phase name="divergent_generation" duration="20 minutes">
Apply techniques in rotation:
- 5 minutes: Mind mapping
- 5 minutes: SCAMPER on top candidates
- 5 minutes: Portmanteau experimentation
- 5 minutes: Foreign language and prefix/suffix exploration
</phase>

<phase name="wild_card" duration="10 minutes">
Deliberately break rules:
- What if the name was a sentence?
- What if it was just a number?
- What if it was misspelled intentionally?
- What if it was from mythology?
</phase>

<phase name="compilation" duration="5 minutes">
Collect all candidates without evaluation. Target: 50+ raw names.
</phase>

</ideation_session_structure>
