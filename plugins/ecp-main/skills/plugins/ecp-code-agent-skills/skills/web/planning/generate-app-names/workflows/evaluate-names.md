# Workflow: Evaluate Names

<required_reading>
**Read these reference files NOW:**
1. references/linguistic-principles.md
2. references/naming-strategies.md
</required_reading>

<process>

## Step 1: Collect Candidates

Gather the names to evaluate. If not provided, ask:
- What are the name candidates to evaluate?
- What is the product/app these names are for?
- Who is the target audience?
- What tone/personality is desired?

## Step 2: Baseline Analysis

For each candidate, document:

**Linguistic Properties:**
- Character count
- Syllable count
- Dominant sounds (plosives, fricatives, vowels)
- Spelling transparency (spelled as sounds?)
- Pronunciation difficulty (1-5 scale)

**Category Classification:**
Identify which naming strategy each represents:
- Descriptive (directly states function)
- Suggestive (hints at benefit)
- Evocative (creates mood/feeling)
- Abstract/Invented (no prior meaning)
- Hybrid/Compound (word combination)
- Acronym/Initialism

## Step 3: SMILE Scoring

Rate each name on the SMILE criteria (1-5 scale):

<scoring_rubric>

**Simple (1-5)**
| Score | Criteria |
|-------|----------|
| 5 | Instantly readable, spellable, sayable by anyone |
| 4 | Clear with minimal effort |
| 3 | Understandable but requires some attention |
| 2 | Confusing spelling or pronunciation |
| 1 | Frequently misspelled or mispronounced |

**Meaningful (1-5)**
| Score | Criteria |
|-------|----------|
| 5 | Immediately communicates brand essence |
| 4 | Clear connection to product benefit |
| 3 | Relevance apparent after brief thought |
| 2 | Connection is tenuous or obscure |
| 1 | No apparent connection to product |

**Imaginative (1-5)**
| Score | Criteria |
|-------|----------|
| 5 | Completely unique and fresh |
| 4 | Distinctive approach, stands out |
| 3 | Competent but not remarkable |
| 2 | Generic or overused pattern |
| 1 | Cliché or boring |

**Lovable (1-5)**
| Score | Criteria |
|-------|----------|
| 5 | Creates genuine delight or emotional response |
| 4 | Pleasant and engaging |
| 3 | Neutral emotional impact |
| 2 | Slightly off-putting |
| 1 | Actively dislikeable |

**Elegant (1-5)**
| Score | Criteria |
|-------|----------|
| 5 | Refined, sophisticated, perfectly appropriate |
| 4 | Well-crafted and fitting |
| 3 | Adequate for purpose |
| 2 | Somewhat clunky or mismatched |
| 1 | Crude, inappropriate, or poorly constructed |

</scoring_rubric>

**Calculate SMILE Total:** Sum of all five scores (max 25)

## Step 4: SCRATCH Deductions

Check each name for SCRATCH issues and deduct points:

<deduction_rubric>

| Issue | Deduction | Check |
|-------|-----------|-------|
| **Spelling challenged** | -3 | Would 8/10 people spell it correctly after hearing it? |
| **Copycat** | -3 | Is it too similar to a well-known brand? |
| **Restrictive** | -2 | Does it limit future product/company growth? |
| **Annoying** | -2 | Is there anything grating about saying or hearing it? |
| **Tame** | -1 | Is it forgettable or bland? |
| **Curse of knowledge** | -2 | Do only insiders understand the reference? |
| **Hard to pronounce** | -3 | Do people stumble when saying it? |

</deduction_rubric>

**Calculate SCRATCH Total:** Sum of applicable deductions

## Step 5: Net Score Calculation

**Net Score = SMILE Total - SCRATCH Deductions**

| Net Score | Rating |
|-----------|--------|
| 20-25 | Excellent candidate |
| 15-19 | Strong candidate |
| 10-14 | Viable with reservations |
| 5-9 | Weak candidate |
| Below 5 | Not recommended |

## Step 6: Comparative Analysis

Create a summary table:

| Name | SMILE | SCRATCH | Net | Strategy Type | Key Strength | Key Concern |
|------|-------|---------|-----|---------------|--------------|-------------|
| Name A | 22 | -2 | 20 | Suggestive | High memorability | Domain taken |
| Name B | 18 | 0 | 18 | Evocative | Unique sound | Harder to spell |
| ... | ... | ... | ... | ... | ... | ... |

## Step 7: Strategic Recommendations

Based on evaluation:

1. **Top recommendation** with rationale
2. **Runner-up options** and when they might be preferred
3. **Names to eliminate** with clear reasons
4. **Suggested modifications** for promising but flawed candidates

Consider factors beyond scores:
- Strategic fit with brand vision
- Competitive differentiation
- Long-term scalability
- Audience resonance

</process>
