<process>

## Step 1: Collect Information

Gather validation requirements:
- What name(s) need validation?
- What is the product category?
- What are the target markets/regions?
- What is the intended use (product name, company name, feature name)?

## Step 2: Trademark Prescreening

**Search these databases for exact and similar matches:**

**United States:**
- USPTO TESS (Trademark Electronic Search System)
- Search exact name and phonetic equivalents
- Filter by relevant Nice Classification classes

**European Union:**
- EUIPO TMview (covers EU + international)
- eSearch plus for detailed trademark information

**Other jurisdictions as relevant:**
- WIPO Global Brand Database (international)
- Country-specific databases for key markets

**Document for each name:**
- [ ] Exact matches found? (Y/N + details)
- [ ] Similar marks in same class? (Y/N + details)
- [ ] Potential conflicts to review with attorney? (Y/N + details)
- [ ] Overall trademark availability assessment (High/Medium/Low risk)

## Step 4: Linguistic Screening

For each target market, check:

**Pronunciation test:**
- Can native speakers pronounce it easily?
- How is it likely to be mispronounced?
- Are there pronunciation variations across dialects?

**Spelling test:**
- Will people spell it correctly after hearing it?
- Common misspellings to anticipate?

**Homophone check:**
- Does it sound like any words in the local language?
- Any slang, profanity, or unfortunate associations?

**Use the checklist from `cultural-considerations.md`:**
- [ ] Pronunciation verified with native speakers
- [ ] Slang/profanity screening completed
- [ ] Homophone risks identified
- [ ] Transliteration options explored (if applicable)

## Step 5: Cultural Evaluation

For each target market, assess:

**Value alignment:**
- Does the name/concept respect local cultural values?
- Any religious sensitivities?
- Historical associations to be aware of?

**Visual/symbolic considerations:**
- If the name implies imagery, is it culturally appropriate?
- Number symbolism issues (4 unlucky in East Asia, etc.)?
- Color associations if brand colors are set?

**Regional specific checks:**
Use the regional considerations from `cultural-considerations.md`:
- Greater China: Tonal considerations, character selection
- Japan: Loan word conflicts, honorific implications
- India: Multi-language considerations
- Middle East: Religious/conservative sensitivity
- Latin America: Spanish/Portuguese variations
- Europe: Multi-national implications

## Step 8: Recommendations

**If low overall risk:**
- Proceed with formal trademark filing
- Secure digital properties
- Begin brand development

**If medium overall risk:**
- Identify specific issues to resolve
- Recommend legal consultation on trademark concerns
- Suggest cultural testing in problematic markets
- Provide mitigation strategies

**If high overall risk:**
- Recommend alternative candidates
- Explain specific blocking issues
- Suggest modifications that might reduce risk
